# Debt Clicker 🎮

An interactive clicker-style game built in **C++** using the **raylib** graphics library.

## 📖 About The Project

Debt Clicker is a small narrative-driven game where the player must repay a debt of **$25,000** by interacting with a mysterious animatronic character running a strange stall. Every click earns money, and players can purchase upgrades to increase their earnings while progressing through the game.

The project was developed as part of a **Fundamentals of Programming** course to demonstrate core programming concepts through game development.

---

## ✨ Features

* 🖱 Mouse-based clicker gameplay
* 🛒 Upgrade & progression system
* 💰 Score tracking and money system
* 💾 File-based save system using `fstream`
* 🎨 Particle effects and simple animations
* 🎬 Multiple game states and cutscenes
* 👾 Final suspense/horror sequence with WASD movement
* 📊 Persistent high score and gameplay data logging

---

## 🕹 Gameplay

### Main Objective

Repay the full **$25,000 debt** by clicking the animatronic character to earn money.

### Upgrades

Players can purchase upgrades to increase money earned per click:

| Upgrade | Cost   | Earnings      |
| ------- | ------ | ------------- |
| Feather | $50    | +1            |
| Needle  | $500   | +5            |
| Hammer  | $3000  | +20           |
| Gun     | $10000 | Special Event |

### Final Event

Purchasing the gun unlocks a hidden sequence where the game changes from a clicker simulator into a suspense-based exploration scene.
